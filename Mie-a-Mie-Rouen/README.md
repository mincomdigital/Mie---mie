# Mie a Mie Rouen

Repository local pret pour le site web Mie a Mie Rouen.

## Demarrage rapide

1. Ouvrir ce dossier dans votre editeur.
2. Ajouter les pages du site (index, menu, contact, etc.).
3. Initialiser Git des que `git` est installe :
   - git init
   - git add .
   - git commit -m "Initial commit"
